# Building Perano from sources

To build Perano language compiler, first you need (obviously) grab sources from the repository.
```shell
git clone https://github.com/noxzion/perano-lang.git
```

If you are on Linux, you will need:
- rust (via rustup)
- build-essential/base-devel
- [chorus](https://github.com/z3nnix/chorus/) (not necessary, but convenient)

If you are on Windows, you will need:
- rust (via rustup)
- Visual Studio Build Tools (DEsktop development with C++)
- Windows 10/11 SDK
- [chorus](https://github.com/z3nnix/chorus/) (not necessary, but convenient)

When you have installed all necessary dependencies, you can build the project:
```shell
cd perano-lang
chorus all
```
